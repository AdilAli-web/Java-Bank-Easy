# Java-Bank-Easy
# 💰 Java Bank Easy

A simple console-based Banking System built using Java and Object-Oriented Programming principles.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 💵 Deposit Money
- 💸 Withdraw Money
- 📊 Check Account Balance
- 📜 Transaction History
- 🖥️ Menu-driven CLI interface

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
  - Encapsulation
  - Abstraction
  - Interfaces
- Clean Code Structure
- Separation of Concerns
- Basic Design Principles

## ⚙️ How to Run

1. Clone the repository:

git clone https://github.com/AdilAli-web/Java-Bank-Easy.git


2. Navigate into the project:

cd Java-Bank-Easy


3. Compile and run:

javac Main.java
java Main
